Cheat Sheet Giải Tích · Casio FX-880BTG · Trắc nghiệm

🖩 Casio: 4 thao tác cốt lõi

① Tích phân xác định bấm được

  • Vào Calculate
  • Nhấn CATALOG → Func Analysis → Integration ∫ → OK
  • Nhập: ∫( hàm số, cận dưới, cận trên )
  • Nhấn EXE → ra số
Ví dụ: ∫(1/(3x+1)², 1, 2) → ra 1/28 ✓
Thể tích xoay Ox: nhập π × ∫([f(x)]², a, b)

② Đạo hàm tại 1 điểm bấm được

  • CATALOG → Func Analysis → Derivative d/dx → OK
  • Nhập: d/dx( hàm số, điểm x₀ )
  • EXE
Đạo hàm riêng f_x tại (x₀,y₀): thay số y₀ vào hàm trước → còn lại hàm 1 biến → bấm d/dx bình thường

③ Hệ phương trình bấm được

  • HOME → Equation → OK
  • Chọn Simultaneous → số ẩn (2 hoặc 3) → OK
  • Nhập hệ số từng phương trình → EXE
Dùng cho: tìm mô hình y=mx+b, tìm điểm tới hạn hàm 2 biến khi hệ tuyến tính

④ Solver – dò nghiệm 1 ẩn bấm được

  • HOME → Equation → Solver → OK
  • Nhập phương trình (dạng f(x) = 0) → EXE
  • Nhập giá trị ban đầu gần nghiệm → chọn Execute → EXE
⚠ Solver chỉ ra 1 nghiệm, phụ thuộc vào giá trị ban đầu. PT có nhiều nghiệm → thử giá trị ban đầu khác nhau.

🎯 Kỹ thuật dò đáp án trắc nghiệm

Dạng "tính giá trị / tìm kết quả" → thay số

Thay x = 1 hoặc x = 2 vào đề bài và 4 đáp án. Đáp án nào cho số trùng khớp → chọn.

Dạng "xác định nguyên hàm" (biết 4 đáp án)

  • Tính ∫( hàm đề, 0, 1 ) → ra số K
  • Thay x=1 vào từng đáp án, trừ giá trị tại x=0
  • Đáp án nào bằng K → đúng

Dạng PT vi phân "nghiệm nào đúng"

Thay đáp án vào PT để kiểm tra: tính y', y'' từ đáp án → thế vào PT → bằng vế phải thì đúng.
Hoặc dùng d/dx tính y' của từng đáp án rồi kiểm tra.

Dạng tìm cực trị / điểm tới hạn

  • Tính d/dx(f, x) → đặt = 0
  • Dùng Solver dò x → được điểm tới hạn
  • Thay vào f để lấy giá trị cực trị

Dạng mô hình tuyến tính y = mx + b

2 điểm dữ liệu → lập 2 PT → vào Equation → Simultaneous → 2 ẩn → ra m, b ngay.

Dạng hàm mũ aᵗ ≤ k

Nhập thẳng vào Solver: a^x - k = 0 → ra t. Không cần lấy logarit tay.

Tìm p: viết mẫu dạng (x-a)ᵖ → đọc p → áp quy tắc.

🗺 Đổi thứ tự tích phân kép suy luận

Quy trình 3 bước – KHÔNG thể bấm máy

  • Đọc cận → viết bất đẳng thức xác định miền D
  • Vẽ phác miền D (chỉ cần phác thảo trên giấy nháp)
  • Đọc lại miền theo thứ tự tích phân ngược

Ví dụ mẫu

Cho: ∫₀¹ ∫ₓ¹ f(x,y) dy dx
Miền D: 0≤x≤1, x≤y≤1 → vẽ ra tam giác
Đổi: x chạy 0→y, y chạy 0→1
→ ∫₀¹ ∫₀ʸ f(x,y) dx dy

Miền hình chữ nhật

Theo Fubini: chỉ hoán đổi cận, giữ nguyên hàm.
∫ₐᵇ ∫꜀ᵈ f dy dx = ∫꜀ᵈ ∫ₐᵇ f dx dy

〰 Tích phân đường suy luận

Nhận loại – nhìn vào ký hiệu đề

Đề cóLoạiCông thức
dsLoại 1∫f·ds = ∫f(x(t),y(t))·√(x'²+y'²) dt
dx, dy riêngLoại 2∫P dx+Q dy = ∫[Px'+Qy'] dt

Khi nào dùng Green thay Loại 2

Đường cong khép kín + miền phẳng → dùng Green:
∮C P dx+Q dy = ∬D (∂Q/∂x - ∂P/∂y) dA
Bước 1: Tính ∂Q/∂x và ∂P/∂y
Bước 2: Tính hiệu → tích phân trên miền D

Tham số hóa đường thẳng A→B

Từ A=(x₁,y₁) đến B=(x₂,y₂):
x = x₁ + t(x₂-x₁), y = y₁ + t(y₂-y₁), t∈[0,1]
dx = (x₂-x₁)dt, dy = (y₂-y₁)dt

📐 Phương trình vi phân suy luận

PT cấp 2 thuần nhất: ay''+by'+cy = 0

Giải PT đặc trưng: ar²+br+c=0 (bấm máy Equation)

  • Nhân 2 vế: (I·y)' = I·Q
  • Tích phân 2 vế → giải ra y
  • Thay điều kiện ban đầu y(x₀)=y₀ → tìm C

dy/g(y) = f(x)dx → tích phân 2 vế độc lập → ra nghiệm
